#Hội Kiến Trúc / Bất động sản

## Setup checklist
- Clone code
- Update env (update database config ...)
- RUN composer install
- RUN php artisan migrate
- RUN npm install
- php artisan ckfinder:download
- mkdir -m 777 public/userfiles
